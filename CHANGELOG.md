# Changelog

## [0.5.0] - 2026-02-01

### Added

- **멀티 게임 지원 (BO5)**
  - 최대 5경기까지 개별 관리
  - 상단 우측에 경기 선택 탭 (1~5)
  - 한 경기에서 픽된 챔피언은 다른 경기에서 선택 불가 (회색 표시)
- **글로벌 밴 시스템**
  - 전체 시리즈에서 밴되는 챔피언 10개 (팀당 5개)
  - 드래그앤드롭으로 밴 등록/해제
- **챔피언 픽 페이지 개선**
  - 진영 교환 버튼 (⇄) 추가
  - 초기화 버튼 - 현재 경기 픽과 글로벌 밴 초기화
- **팀 설정 페이지 개선**
  - 진영 색깔 제거 (팀 A, 팀 B로 중립화)
  - 팀별 비우기 버튼 추가

### Changed

- 상태 구조 변경: 게임별 챔피언 정보 분리
- 팀 설정에서 진영 교환 버튼 제거 (챔피언 픽에서만 가능)

- 상세 정보에 역할 표시 추가
