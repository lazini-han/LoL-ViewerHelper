# 리그 오브 레전드 대회 시청 도우미 웹 사이트 제작 프로젝트

## 프로젝트의 목적
- 리그 오브 레전드의 각 챔피언들이나 오브젝트, 아이템들에 대한 지식이 부족한 시청자들이 경기를 더 재밌게 이해할 수 있게 돕는다.
- 컴퓨터 영상으로 경기를 보다가 다른 브라우저 창에 도우미 사이트를 띄워서 편리하게 궁금한 것을 빠르게 확인할 수 있게 한다.

## 지식이 부족한 시청자가 경기를 시청할 때 발생하는 문제점
1. 중계 방송에서 "챔피언 이름"과 "플레이어 닉네임"을 섞어서 사용하는데, 챔피언 이름을 모르기 때문에 설명을 이해하지 못하거나 파악하는데 오래 걸린다.
2. 각 챔피언이 어떤 역할에 특화되어있는지 모르기 때문에, 현재 상황이 해당 플레이어가 잘하고 있는건지 아닌지 설명할때 이해를 못한다.
3. 프로 게이머가 한타 싸움 같이 정교하게 서로 기술을 사용하는 장면에서, 화면에 보이는 어떤 효과가 어떤 챔피언이 사용하는 기술인지, 어떤 효과가 있는지 모르고 보게 된다.
4. 중계인이 챔피언들이 사용하는 아이템에 대해서 언급할때, 어떤 아이템인지 아이콘도 모르고, 그 아이템이 챔피언을 어떻게 변화시키는지 몰라서 상황 변화를 파악하지 못한다.
5. 게임 중 발생하는 주요 오브젝트들이 승패에 어떤 식으로 언제 영향을 주는지 몰라서 경기의 흥미로는 점을 파악하지 못한다.

## 현재 사용중인 방법
- 모든 챔피언의 한국어 이름, 영문 이름, 썸네일 아이콘 이미지 URL 정보를 바탕으로 구글 시트를 이용하고 있음.

- 경기가 시작될 때, 밴 픽 단계에서 양팀이 고른 챔피언 10개를 전체 챔피언 목록에서 체크한다.
- 각 팀 이름과 선수 닉네임, 포지션을 시트에 입력한다.
- 해당 경기의 팀별 진영을 선택하고, 각 선수에게 앞서 체크한 10개의 챔피언을 할당한다.
- 그 결과, 총 10명의 선수의 챔피언 이름, 닉네임, 포지션을 한 눈에 파악할 수 있는 시트를 옆에 띄워두고 영상을 시청한다.

## 구축할 웹사이트에 대한 고려사항
- GitHub 을 구독중이라는 점을 이용한다.
- AI를 통해 제작을 진행한다.
- 외부 DB를 사용하지 않는다.
- DB를 활용한다면, 구글 스프레드 시트 혹은 json 파일 등을 활용한다.
- GitHub Pages 기능으로 서비스할 수 있도록 제작한다.

## 1단계 MVP : 현재 사용중인 방법을 웹사이트에서 발전된 방식으로 구현
우선 사용자는 다음과 같은 방식으로 경기를 설정할 것입니다.

1. 팀 설정 페이지 : 각 팀의 이름 입력, 선수 이름 입력 후, 선수별 포지션을 선택합니다.
   - 추후 팀별 선수 정보도 저장된 것을 사용할 수 있도록 하는 기능을 추가 고려. 현재는 제외.

2. 챔피언 픽 페이지
   1. 페이지 상단에는 모든 챔피언의 한글 이름과 아이콘 이미지를 표로 한눈에 보여준다. 
   2. 하단에는 왼쪽에는 블루팀, 오른쪽에는 레드팀이 선택한 챔피언 5명의 아이콘이 들어갈 공간을 보여준다. 두 팀 사에 가운데에는 양팀의 팀 이름이 "vs" 마크를 끼고 보인다.
   3. 블루팀은 왼쪽부터 서포터(5), 원딜(4), 미드(3), 정글(2), 탑(1) 순서고, 레드 팀은 반대로 탑(1), 정글(2), 미드(3), 원딜(4), 서포터(5) 순서다. 여기에는 팀 설정에서 설정한 선수 이름들이 아이콘이 들어갈 공간 하단에 표시된다.
   4. 원하는 챔피언을 상단에서 아이콘으로 클릭해서 드래그해서 원하는 하단 칸 빈 곳에 놓을 수 있다.
      a. 선택된 챔피언은 상단 페이지에서는 해당 챔피언 아이콘은 그대로 있지만, 하단 칸 어디에 해당 챔피언이 놓였냐에 따라서 팀 색깔(블루 진영 또는 레드 진영)이 배경으로 표시되고, 더이상 드래그 할 수 없다. 
      b. 놓여진 하단 칸에는 챔피언 아이콘이 보이고 아이콘 아래에는 선수 이름과 함께 챔피언 이름도 보인다.
      c. 하단 칸에 챔피언 아이콘을 드래그해서 상단 쪽 아무 곳에 놓으면, 챔피언이 하단 공간에서 비워지고 a에서 표시했던 색깔도 사라지고, 다시 드래그 할 수 있게 된다.
      d. 하단 칸에 챔피언 아이콘을 다른 하단 칸으로 드래그 해서 놓을 수 있다. 빈 칸이었으면 챔피언이 그 칸으로 이동하고, 다른 챔피언이 차였던 칸이었으면 서로의 챔피언이 교환된다.

3. 경기 상황 페이지
2에서 선택된 각 챔피언을 팀 별로 포지션에 따라서 세로로 배치한다.
포지션 순서대로 왼쪽엔 블루팀, 오른쪽엔 레드팀이 표시된다.
가장 위에는 팀 이름이 있고, 아래로 챔피언 아이콘, 챔피언 이름, 선수 이름이 표시된다.
둘 사이에는 포지션을 묘사하는 아이콘들이 들어간다.
이는 최대한 여백을 적게 해서 큰 화면으로 영상을 볼 때 옆에 매우 좁은 가로 폭으로 띄워둬도 방해되지 않도록 한다.
